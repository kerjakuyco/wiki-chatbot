Super hard to find documentation in Shield?

### An assistant who will be
- Stand by 24/7
- Provide direct answer

### How to

#### Install All Required Module

```
  npm install
```

#### Setup .env

```
  cp .env.example .env
```

#### Run App

```
  npm run start
```
